# Desafio Contador  

Este repositório contém a solução para o **Desafio Contador**, proposto no módulo **Sintaxe Básica com Java** do **Bootcamp Bradesco** em parceria com a **DIO (Digital Innovation One)**.  

## Tecnologias Utilizadas  

- Java  
- IDE de sua escolha (Eclipse, IntelliJ, VS Code etc.)  
- Git/GitHub  

## Descrição do Desafio  

O objetivo deste desafio é praticar os conceitos fundamentais de **sintaxe Java**, incluindo:  

✅ Estruturas de controle de fluxo (`if`, `else`, `loops`)  
✅ Entrada e saída de dados  
✅ Manipulação de variáveis e operadores  
✅ **Tratamento de exceções com `try-catch` e uma exceção personalizada**
